# Specification for I/O-shield input

## General
All buttons and switches are read from right to left (4 3 2 1)

## Buttons

1. Transpose.
2. Clear
3. Undo
4. Play/Pause

## Switches

1. Nothing
2. Transpose control (Up/Down)
3. Record enable
4. Metronome enable

## Potentiometer
Controls tempo. Turn clockwise to increase and counter-clockwise to decrease.
